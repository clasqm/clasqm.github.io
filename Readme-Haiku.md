# Software for the Haiku Operating System

Most of these were written in the [yab](www.yab.orgfree.com) language. I'm not really working on this any more. These are now obsolete or no longer pique my interest. Feel free to steal any code you see from here.

+ [TinyTim](https://clasqm.github.io/TinyTim/) - GUI frontend to the tinyurl.com URL-shortening website. - *beta*
+ [Trope](https://clasqm.github.io/Trope2/) - Text Editor for Haiku - *beta*
+ [Rondel](https://clasqm.github.io/Rondel/) - Markdown-aware text editor for Haiku with HTML-generation - *beta*
+ [barberpole](https://clasqm.github.io/barberpole/) - Display a spinning barberpole with a message - *beta*
+ [FreeVerse](https://clasqm.github.io/FreeVerse/) - A notepad for Appointments, To-dos and Diary entries - *beta*
+ [FuzzyTime](https://clasqm.github.io/FuzzyTime/) - Haiku desktop applet to display the date and time in 'fuzzy' format - *beta*
+ [libclasqm](https://clasqm.github.io/libclasqm/) - Clasqm's libraries for yab - *beta*
+ [LiftOff](https://clasqm.github.io/LiftOff/) - An application launcher for the Haiku OS - *beta*
+ [makehaiku](https://clasqm.github.io/makehaiku/) - CLI utlity to write haikus - *ready to compile, but the list of terms needs more entries*
+ [My_PDFs](https://clasqm.github.io/My_PDFs/) - PDF document collection manager - *beta*
+ [setpermissions](https://clasqm.github.io/setpermissions/) - File permission manipulation utility for Haiku - *beta*
+ [TextLab](https://clasqm.github.io/TextLab/) - GUI front-end to iconv text conversion utility
+ [Xtract1](https://clasqm.github.io/Xtract1/) - Single file .zip extractor.
+ [Yabadabbadoo](https://clasqm.github.io/Yabadabbadoo/) - An alternative IDE for the yab programming language.
+ [yabdialog](https://clasqm.github.io/yabdialog/) - A CLI utility that displays a variety of dialogs, alerts and panels.
+ [ZipDropper](https://clasqm.github.io/ZipDropper/) - A zip utility with a minimalist interface.
+ [crashtest](https://clasqm.github.io/crashtest/) - Script offers disk check if computer crashed.
+ [BAS 4 Haiku](https://clasqm.github.io/bas4haiku/) - [hosted]
+ [FontMonkey](https://clasqm.github.io/FontMonkey/) - program to preview, load and unload fonts from the clasqm repo.
+ [GraphicsWorkflow](https://clasqm.github.io/GraphicsWorkflow/) - A system to string various graphics applets into a sequence, a bit like Automator on Mac OSX - *I may strip out the individual utlilities from this*.
+ [MenuBar](https://clasqm.github.io/MenuBar/) - A menu for the Haiku Desktop.
+ [References](https://clasqm.github.io/References/) - A reference manager for Haiku - *pre-alpha*
+ [ScriptAssistant](https://clasqm.github.io/ScriptAssistant/) - A little utility to display the help content of scripting commands.
+ [vstm-gui](https://clasqm.github.io/vstm-gui/) - Simple TODO list manager for Haiku with graphical front-end.
+ [yabadabbadoo github_integration](https://clasqm.github.io/yabadabbadoo_github_integration/) - Github integration scripts for Yabadabbadoo.
+ [yabTemplatePicker](https://clasqm.github.io/yabTemplatePicker/) - A template picking accessory app for the Yabadabbadoo IDE. Also usable as a standalone app.
+ [yabTemplatePicker templates](https://clasqm.github.io/yabTemplatePicker_templates/) - A set of templates for the [yabTemplatePicker](https://github.com/clasqm/yabTemplatePicker) app.
