# Software by Michel Clasquin-Johnson on Github

## Software for Linux

+ [Yabasic](https://clasqm.github.io/yabasic/) Framework to build a Debian package of yabasic. - *ongoing*
+ [Yabasic libraries](https://clasqm.github.io/yabasiclibs/) - Extension libraries for yabasic. - *alpha*


## OS-Independent software

+ [Belief](https://clasqm.github.io/Belief/) - A CLI utility to generate random religious creeds. A bit like fortune. - *[hosted] ready to compile on anything with a C compiler.*


## Software for Haiku OS

Most of these were written in the [yab](www.yab.orgfree.com) language. I'm not really working on this any more - at least not until Haiku reaches a stable beta status. Apps marked "*beta*" below should compile and run well enough. I am just not ready to call anything a Release until I know the platform they run on will be relatively stable.

+ [barberpole](https://clasqm.github.io/barberpole/) - Display a spinning barberpole with a message - *beta*
+ [crashtest](https://clasqm.github.io/crashtest/) - Script offers disk check if computer crashed - *beta*
+ [BAS 4 Haiku](https://clasqm.github.io/bas4haiku/) - *[hosted] ready to compile*
+ [FontMonkey](https://clasqm.github.io/FontMonkey/) - program to preview, load and unload fonts from the clasqm repo. - *beta*
+ [FreeVerse](https://clasqm.github.io/FreeVerse/) - A notepad for Appointments, To-dos and Diary entries - *beta*
+ [FuzzyTime](https://clasqm.github.io/FuzzyTime/) - Haiku desktop applet to display the date and time in 'fuzzy' format - *beta*
+ [GraphicsWorkflow](https://clasqm.github.io/GraphicsWorkflow/) - A system to string various graphics applets into a sequence, a bit like Automator on Mac OSX - *pre-alpha*
+ [libclasqm](https://clasqm.github.io/libclasqm/) - Clasqm's libraries for yab - *beta*
+ [LiftOff](https://clasqm.github.io/LiftOff/) - An application launcher for the Haiku OS - *beta*
+ [makehaiku](https://clasqm.github.io/makehaiku/) - CLI utlity to write haikus - *ready to compile, but the list of terms needs more entries*
+ [MenuBar](https://clasqm.github.io/MenuBar/) - A menu for the Haiku Desktop - *obsolete*
+ [My_PDFs](https://clasqm.github.io/My_PDFs/) - PDF document collection manager - *beta*
+ [Rondel](https://clasqm.github.io/Rondel/) - Markdown-aware text editor for Haiku with HTML-generation - *beta*
+ [References](https://clasqm.github.io/References/) - A reference manager for Haiku - *pre-alpha*
+ [ScriptAssistant](https://clasqm.github.io/ScriptAssistant/) - A little utility to display the help content of scripting commands - *beta*
+ [setpermissions](https://clasqm.github.io/setpermissions/) - File permission manipulation utility for Haiku - *beta*
+ [TextLab](https://clasqm.github.io/TextLab/) - GUI front-end to iconv text conversion utility - *beta*-
+ [TinyTim](https://clasqm.github.io/TinyTim/) - GUI frontend to the tinyurl.com URL-shortening website. - *beta*
+ [Trope](https://clasqm.github.io/Trope/) - Text Editor for Haiku - *beta*
+ [vstm-gui](https://clasqm.github.io/vstm-gui/) - Simple TODO list manager for Haiku with graphical front-end - *beta*
+ [Xtract1](https://clasqm.github.io/Xtract1/) - Single file .zip extractor - *beta*
+ [Yabadabbadoo](https://clasqm.github.io/Yabadabbadoo/) - An alternative IDE for the yab programming language - *beta*
+ [yabadabbadoo github_integration](https://clasqm.github.io/yabadabbadoo_github_integration/) - Github integration scripts for Yabadabbadoo - *beta*
+ [yabdialog](https://clasqm.github.io/yabdialog/) - A CLI utility that displays a variety of dialogs, alerts and panels  - *beta*
+ [yabTemplatePicker](https://clasqm.github.io/yabTemplatePicker/) - A template picking accessory app for the Yabadabbadoo IDE. Also usable as a standalone app. - *beta*
+ [yabTemplatePicker templates](https://clasqm.github.io/yabTemplatePicker_templates/) - A set of templates for the [yabTemplatePicker](https://github.com/clasqm/yabTemplatePicker) app. - *beta*
+ [ZipDropper](https://clasqm.github.io/ZipDropper/) - A zip utility with a minimalist interface - *beta*
