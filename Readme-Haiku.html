Software for the Haiku Operating System

Most of these were written in the yab language. I’m not really working on this any more. These are now obsolete or no longer pique my interest. Feel free to steal any code you see from here.

  • TinyTim - GUI frontend to the tinyurl.com URL-shortening website. - beta
  • Trope - Text Editor for Haiku - beta
  • Rondel - Markdown-aware text editor for Haiku with HTML-generation - beta
  • barberpole - Display a spinning barberpole with a message - beta
  • FreeVerse - A notepad for Appointments, To-dos and Diary entries - beta
  • FuzzyTime - Haiku desktop applet to display the date and time in ‘fuzzy’ format - beta
  • libclasqm - Clasqm’s libraries for yab - beta
  • LiftOff - An application launcher for the Haiku OS - beta
  • makehaiku - CLI utlity to write haikus - ready to compile, but the list of terms needs more entries
  • My_PDFs - PDF document collection manager - beta
  • setpermissions - File permission manipulation utility for Haiku - beta
  • TextLab - GUI front-end to iconv text conversion utility
  • Xtract1 - Single file .zip extractor.
  • Yabadabbadoo - An alternative IDE for the yab programming language.
  • yabdialog - A CLI utility that displays a variety of dialogs, alerts and panels.
  • ZipDropper - A zip utility with a minimalist interface.
  • crashtest - Script offers disk check if computer crashed.
  • BAS 4 Haiku - [hosted]
  • FontMonkey - program to preview, load and unload fonts from the clasqm repo.
  • GraphicsWorkflow - A system to string various graphics applets into a sequence, a bit like Automator on Mac OSX - I may strip out the individual utlilities from this.
  • MenuBar - A menu for the Haiku Desktop.
  • References - A reference manager for Haiku - pre-alpha
  • ScriptAssistant - A little utility to display the help content of scripting commands.
  • vstm-gui - Simple TODO list manager for Haiku with graphical front-end.
  • yabadabbadoo github_integration - Github integration scripts for Yabadabbadoo.
  • yabTemplatePicker - A template picking accessory app for the Yabadabbadoo IDE. Also usable as a standalone app.
  • yabTemplatePicker templates - A set of templates for the yabTemplatePicker app.
