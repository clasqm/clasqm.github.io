Software for the Haiku Operating System

Most of these were written in the yab language. I'm not really working on this any more - at least not until Haiku reaches a stable beta status. Apps marked "beta" below should compile and run well enough. I am just not ready to call anything a Release until I know the platform they run on will be relatively stable.

UPDATE: Haiku has reached beta status, and I am starting to do ground-up rewrites of my apps. It will take a while, though.

Pre-beta apps

  • barberpole - Display a spinning barberpole with a message - beta
  • crashtest - Script offers disk check if computer crashed - beta
  • BAS 4 Haiku - [hosted] ready to compile
  • FontMonkey - program to preview, load and unload fonts from the clasqm repo. - beta
  • FreeVerse - A notepad for Appointments, To-dos and Diary entries - beta
  • FuzzyTime - Haiku desktop applet to display the date and time in 'fuzzy' format - beta
  • GraphicsWorkflow - A system to string various graphics applets into a sequence, a bit like Automator on Mac OSX - pre-alpha
  • libclasqm - Clasqm's libraries for yab - beta
  • LiftOff - An application launcher for the Haiku OS - beta
  • makehaiku - CLI utlity to write haikus - ready to compile, but the list of terms needs more entries
  • MenuBar - A menu for the Haiku Desktop - obsolete
  • My_PDFs - PDF document collection manager - beta
  • Rondel - Markdown-aware text editor for Haiku with HTML-generation - beta
  • References - A reference manager for Haiku - pre-alpha
  • ScriptAssistant - A little utility to display the help content of scripting commands - beta
  • setpermissions - File permission manipulation utility for Haiku - beta
  • TextLab - GUI front-end to iconv text conversion utility - beta-
  • TinyTim - GUI frontend to the tinyurl.com URL-shortening website. - beta
  • Trope - Text Editor for Haiku - beta
  • vstm-gui - Simple TODO list manager for Haiku with graphical front-end - beta
  • Xtract1 - Single file .zip extractor - beta
  • Yabadabbadoo - An alternative IDE for the yab programming language - beta
  • yabadabbadoo github_integration - Github integration scripts for Yabadabbadoo - beta
  • yabdialog - A CLI utility that displays a variety of dialogs, alerts and panels - beta
  • yabTemplatePicker - A template picking accessory app for the Yabadabbadoo IDE. Also usable as a standalone app. - beta

  • ZipDropper - A zip utility with a minimalist interface - beta

Post-beta apps

Existing applications will be rewritten from the ground up. The idea here is to reduce the reliance on third-party applications where possible and fix some things that I didn't quite get right the first time. The resulting apps will have the same internal name as before (e.g. Trope rather than Trope 2).
